# LUMINÉ – Radiant Skin App

A premium, high-performance skincare e-commerce demonstration application. LUMINÉ showcases modern web development practices with a focus on immersive design, fluid animations, and a seamless user experience.

## ✨ Features

- **Immersive Animations:** Utilizes GSAP (GreenSock) for smooth, scroll-triggered animations and parallax effects.
- **Modern UI Architecture:** Built with React 19 and styled with Tailwind CSS v4 for rapid, responsive design.
- **Accessible Components:** Integrates Radix UI primitives for accessible, unstyled UI components (Dialogs, Accordions, Tabs, etc.).
- **Server-Side Rendering (SSR):** Powered by TanStack Start for lightning-fast initial page loads and excellent SEO.
- **Type-Safe Routing:** Leverages TanStack Router for fully type-safe navigation and state management.

## 🛠️ Tech Stack

- **Framework:** [React 19](https://react.dev/) + [TanStack Start](https://tanstack.com/start/latest)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **UI Components:** [Radix UI](https://www.radix-ui.com/) + [Lucide React](https://lucide.dev/) (Icons)
- **Animations:** [GSAP](https://gsap.com/)
- **Build Tool:** [Vite](https://vitejs.dev/)

## 🚀 Getting Started

### Prerequisites

Make sure you have Node.js and npm (or bun) installed on your machine.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/MercySpectures/customLogicInnovation-Lumine.git
   ```

2. Navigate to the project directory:
   ```bash
   cd radiant-luminescence-project
   ```

3. Install the dependencies:
   ```bash
   npm install
   ```

### Running Locally

To start the development server:

```bash
npm run dev
```

The application will be available at `http://localhost:8080`.

### Building for Production

To create a production build:

```bash
npm run build
```

You can preview the built application locally by running:

```bash
npm run preview
```
